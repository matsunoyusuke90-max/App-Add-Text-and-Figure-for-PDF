# PDFにテキスト書き込み

PDF にテキストや図形を重ねて書き込み、注釈付きPDFとして書き出せるブラウザアプリです。プロジェクト単位で複数のPDFを管理できます。
ダウンロードして個人用にお使いください。

## 使い方

`index.html` をブラウザで開くだけ。

1. ホーム画面で「新規プロジェクト」を作成し、PDFを読み込む。
2. 編集画面でテキスト・図形を追加し、ドラッグで配置・トリミング・複製・塗りつぶしなどを行う。
3. 編集済みPDFをエクスポート（ダウンロード）する。

## 機能

- **プロジェクト管理** — ホーム画面にプロジェクトをカード表示。開く／名前変更／削除／複製に対応。
- **テキスト・図形の書き込み** — ドラッグ配置、フォントサイズ調整、塗りつぶし、トリミング。
- **Undo / Redo** — 編集操作の取り消し・やり直し。
- **ダークモード** — ライト／ダークの切り替え（`data-theme`）。
- **日本語フォント埋め込み** — 日本語テキストを正しく出力。

## データの保存

- プロジェクトとPDFデータはブラウザの **IndexedDB** に保存されます（この端末のみ）。
- 一部の設定は **localStorage** に保存されます。
- サーバーには送信されません。ブラウザのデータを消去すると保存内容も消えます。

## 技術メモ

- **pdf-lib** でPDFを生成・書き込み、**pdfjs-dist (pdf.js)** で表示。
- 日本語グリフ欠落を避けるため、フォント埋め込みは **`subset:false`** で行っています（`subset:true` だと日本語が欠ける）。
- 日本語フォントは同梱の `japanese-400-normal.ttf` を使用。

## ファイル構成

| パス | 内容 |
|---|---|
| `index.html` | 本体（これを開く） |
| `japanese-400-normal.ttf` | 埋め込み用の日本語フォント |
| `sample.pdf` | 動作確認用のサンプル |
| `package.json` / `package-lock.json` | 依存定義（pdf-lib, pdfjs-dist など） |
| `node_modules/` | 依存パッケージ |
| `データ確認.html` / `保存テスト.html` | 開発用の確認ツール |

> `package.json` の依存はビルド／検証用です。`index.html` 単体はライブラリをCDN等から読み込んで動作します。
